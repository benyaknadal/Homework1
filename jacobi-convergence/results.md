1️⃣:  نتائج الاختبار

## مثال 1️⃣: مصفوفة متقاربة (Convergent Matrix)

### المصفوفة

```matlab
A1 = [4  -1   0;
     -1   4  -1;
      0  -1   4];
```

### الاختبارات

```matlab
>> is_diagonally_dominant(A1)
ans =
     1    % true - المصفوفة تحقق السيطرة القطرية الصارمة

>> [B1, rho1] = jacobi_spectral_radius(A1)
rho1 =
    0.5000
```

### التحليل

* السيطرة القطرية: **متحققة ✓**

  * الصف 1: ( |4| > |-1| + |0| = 1 )
  * الصف 2: ( |4| > |-1| + |-1| = 2 )
  * الصف 3: ( |4| > |0| + |-1| = 1 )

* نصف قطر الطيف:

[
\rho = 0.5 < 1
]

✔ النتيجة: الطريقة تتقارب **بسرعة معقولة**

#  الاستنتاجات

* **المعيار الحاسم للتقارب:**

[
\rho(B) < 1
]

* السيطرة القطرية:

  * تضمن التقارب ✔
  * لكنها **ليست شرطاً ضرورياً**

* **سرعة التقارب**:

  * تتناسب عكسيًا مع قيمة ( \rho(B) )

* **قاعدة عملية**:

  * كلما كان ( \rho(B) ) أصغر، كان التقاربالمللاستخدام داخل ملف المشروع مباشرة.
